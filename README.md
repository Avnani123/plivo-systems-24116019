# Real-Time UDP Audio Playout Engine

A robust, low-latency audio transport and playout engine designed to handle network jitter, packet drops, reordering, and duplication over UDP.

## 📦 Deliverables Checklist
- sender.c & receiver.c: Core media transport & FEC handling
- Makefile: Builds binary executables ./sender and ./receiver
- SUMMARY.html: In-depth architectural documentation
- RUNLOG.md: Experimentation logs and performance metrics
- NOTES.md: System design choices and evaluation parameters
